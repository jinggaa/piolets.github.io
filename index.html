Soal 1

1. Mengapa menggunakan CSS dan JavaScript dalam pemrograman web?

CSS merupakan singkatan dari Cascanding Style Sheet yang dimana memungkinkan pengembang untuk memisahkan struktur (HTML) dari tampilan (CSS) dalam pengembangan web. CSS ini digunakan untuk mengubah tampilan dan tata letak elemen di html, dapat mengatur warna, ukuran teks, font , jarak antar elemen, dan banyak aspek lain dari desain halaman web.

JavaScript dalam sebuah web digunakan untuk membuat efek animasi, validasi formulir, pemrosesan data, membuat halaman web menjadi interaktif dan dinamis.

Soal 2

2. Bagaimana CSS dan JavaScript dapat diintegrasikan dalam pemrograman web

Pada CSS ada tig acara integrasikan ke web, yaitu :

  • ditulis langsung pada style elemen html
  • ditulis dalam tag style dibagian head html
  • file css nya dipisah dan nanti dipanggil menggunakan link

Untuk javascript juga ada beberapa cara untuk diintegrasikan pada web, yaitu :

  • ditulis langsung pada elemen html
  • ditulis pada tag script pada file html, biasanya ini pada akhir bagian body
  • diletakkan pada file .js nanti dipanggil dengan tag script src =” “

Soal 3

3. Jelaskan keuntungan menggunakan javascript dan css

Pada pemrograman web dalam menggunakan java dan css memiliki keuntungannya masing-masing, keuntungan menggunakan css yaitu :

  1. memisahkan konten dan tampilan
  2. konsistensi tampilan, css bisa digunakan untuk banyak halaman
  3. memungkinkan web ditampilkan dengan baik
  4. halaman berikutnya lebih cepat dan mudah diakses
  5. mengurangai beban server

Keuntungan menggunakan javascript pada web yaitu :

  1. memungkinkan halaman dapat merespon pengguna
  2. mengambil dan mengirim data ke server
  3. bisa melakukan banyak proses pada sisi pengguna
